1- Les fonctions prédéfinies

Les fonctions prédefinies

Une fonction est un bloc de code nommé et réutilisable, et dont le but est d'éffectué et dont le but est d'effectué une tache précise

Le nom d'une fonction est suivi d'un couple de parenthèses qui contiennent le plus souvent des arguments ex. replace()

le premier argument de replace est la chaine de caractères que l'on recherche et le second est celle qui va la remplacer.

Les fonctions personalisées

Exemples

A- ex. randomX100()

Le premier ex de la création de fonctiona pour but de nous montrer que l'on peut éxécuter une fonction à l'intérieur d'une autre fonction

function randomX100(){
return Math.random() * 100; }

Récapitulatif sur les fonctions

Les fonctions sont des blocs de code nommé et réutilisable et dont le but est d'effectuer une tâche précise

  • Il existe 2 types de fonctions en JS : les fonctions natives ou prédefinies (ou méthodes )qu'on aura qu'appeler et les fonctions personalisées qu'on va créer
  • Pour exécuter le code d'une fonction il faut l'appeler. Pour cela, il suffit d'écrire son nom suivi de parenthèses en passant éventuellement des arguments dans les parenthèses.
  • On crée une fonciton personnalisée grâce au mot-clef "function" :
    • : si une fonction a besoin qu'on lui passe des valeurs pour fonctionner, alors on définira des paramètres
    • lors de sa définition. place des paramètres
