JS - III Les structures de contrôle

1- Les conditions et les boucles

Il existe deux types de structures de contrôle de base, que l'on retrouve dans tous les languages de programmations, et donc JS.

  • Les structures de contrôle conditionnelles : les conditions
  • Les structures de contôle de boucles : les boucles

Les conditions vont nous permettre d'éxécuter un certain nombre d'instruction si seulement si et seulement si une certaine condition est vérifiée

Les boucles vont permettre d'éxécuter un bloc de code tant qu'une condition donnée est vérifiée

A- Présentation des conditions

  • La condition if (si) ;
  • La condition if ... else (si ...non);
  • La condition if ... elseif... else (si ... sinonsi... sinon
