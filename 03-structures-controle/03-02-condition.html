JS - III 02 Condition (IF)

1- if ...

lA condition if est l'une des plus utilisées et est également la plus simple à comprendre puisqu'elle va nous permettre d'éxécuté un bloc de code SI et SEULEMENT SI le résultat vaut true

2 If éxécuté dans le fichier "condition.html"

  • if (x> 1){// la condition ou le test est évalué à : "true" alert('x contient une valeur supérieur à 1');// si c'est "true" on éxécute ce code. Si c'est "false", on n'éxécute pas l'alerte }
  • if(x==y) {// test si x et y contiennent la même valeur. La réponse estt non alert('x et y contiennent la même valeur'); }
  • if(y){ alert('la valeur de y est évalué à: "true'); }

!! Dans ce 3e test d'une condition va être évalué dans le texte booléen ; donc à trueà l'exception des valeurs suivantes qui vont être évalauées à false

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)
