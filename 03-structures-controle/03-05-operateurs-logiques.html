JS - III 05 Les Opérateurs logiques

Ce type d'opérateurs va nous permettre d'éffectuer plusieurs comparaisons dans nos test ou d'inverser la valeur logique du résultat d'un test

1-Présentation et utilisation des opérateurs logiques

Le JS supporte 3 opérateurs logiques : l'opérateur "ET", "ou", et "non"


Opérateurs logiques
Opérateurs,nom Opérateurs, Symboles Descriptions
AND(ET) && renvoi true si toutes les comparaisons sont évalués à true ou false sinon
OR(ou) || renvoie true si au moins l'une des comparaisons est évaluée à true ou encore false sinon
NO(NON) ! Renvoie false si une comparaison est évaluée à true ou renvoie true dans le cas contraire
