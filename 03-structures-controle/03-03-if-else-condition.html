JS - III 03 if else Condition (IF)

1- if else ... else en JS

La structure conditionnelle if ... else sera plus complète car on va exécuter un bloc de code si un test renvoie trueou un autre dans le cas contraire

If ... else ...

  • if (x>1){
    alert('x contient une valeur supérieur à 1');
    }else{
    alert('x contient une valeur inférieur à 1');
    }
