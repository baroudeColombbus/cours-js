JS - III 04 if else if else Condition

1- if else ... if ... else en JS

La condition if ... else .. if ... else (si ... non ...si ...non) est une structure conditionnelle encore plus complète puisqu'elle permet de générer et de prendre en charge autant de cas que l'on souhaite

Nous allons pouvoir écrire autant de else if que l'on veut dans cette condition

Example: if(x > 1){
alert('x contient une valeur strictement supérieur à 1');
}else if(x ==1){
alert('x contient la valeur égal a 1');
}else{
alert('x contient une valeur est strictement inférieur à 1 '); }

Dès qu'un test va être validé je JS va ignorer les tests suivant. Il faut donc faire attention à l'ordre des else lors de l'écriture d'une condition pour obtenir le résultat souhaité.

Exemple : if(x > 1){//si
alert('x contient une valeur strictement supérieur à 1');
}else if(x == 1){//sinon si
alert('x contient la valeur 1');
}else if(x < 1){
alert('x est strictement inférieur à 1');
}else if(x < 0){
alert('x contient strictement inférieur à 0'); }else if(x < -2){
alert('x contient strictement inférieur à -2');
}else{
alert('la valeur de x n\'a pas pu être comparée à un nombre'); }
