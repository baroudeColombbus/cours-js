JS - III 10 for

La boucle for (pour en français) est différente strucurellement de while... et de do..., puis qu'on initialise la variable à l'intérieur de la boucle

Notez qu'on utilise généralement la lettre « i » (pour « iterator ») dans les boucles en général et particulièrement au sein des boucles for pour les reconnaitre plus rapidement dans un script. Cependant, ce n'est pas obligatoire et vous pouvez utiliser n'importe quel autre nom de variable.

Table

L'instruction "break"

Dans les paragraphes qui sont crées ci-dessous à l'aide d'une boucle for nous inserons un code if... à l'extérieur de la boucle ; le if test une condition et la contient une "instruction", break qui stoppe la boucle quand la condition est vrai

Exemple: for(let arreteToi = 0; arreteToi < 100; arreteToi++){
if(arreteToi == 62){
break;
} document.getElementById('p2').innerHTML += "arreteToi contient la valeur " + arreteToi + ' à chaque passage de la boucle for.
' }

L'instruction "continue"

Avec cette instruction on peut "sauter" une "itération" : dans notre exemple on n'affiche que les itérations qui sont paires. voir. les commentaires dans le fichier for.js

Exemple
for(let item = 0; item < 10; item++ ){
if(item % 2 != 0){
continue;
}
document.getElementById('p3').innerHTML = 'la variable item contient ' + item + ' à chaque passage de la boucle.
' }
