JS - IX 01 - Le DOM

Le DOM : Document Object Model

Définition

1- Le DOM est une interface de programmation pour les document HTML ou XML qui représente le document (la page actuelle) sous une forme qui permet aux languages de script comme javaScript d'y accéder et d'en manipuler le contenu et les styles.
